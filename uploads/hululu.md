# NEXORA AI — design.md

## Project Vision
Build a premium SaaS landing page that feels crafted by a world-class product team.
The design language is clean, bold, minimal, architectural, and product-first.

## Core Principles
- Less text, more product.
- Large editorial typography.
- Generous whitespace.
- Perfect alignment.
- Calm motion.
- Product UI is the hero.
